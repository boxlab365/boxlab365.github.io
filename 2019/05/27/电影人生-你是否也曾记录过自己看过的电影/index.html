电影人生-你是否也曾记录过自己看过的电影

发表于 2019-05-27 | 分类于 生活
本文字数： 570 | 阅读时长 ≈ 1 分钟
一部电影，看完你是否曾想过表达什么？亦或者记录一些什么？


不想大多数人一样，使用国内的影视平台，去看影视
而手机也几乎不曾安装过…
不过，这并不代表我不爱看电影（貌似对电视剧不感兴趣）
如较真的话，想想可能就是：青盲（越狱题材）、亮剑、水浒传、西游记
当然，还有一些动漫（发现偏向日漫）如:进击的巨人….
说实话，看了也不少，能想起来的，还真不是很多
不过个人发现了一个规律：动漫偏向日&美（迪士尼）战争&大片题材偏向美、武林&古装题材偏向中;当然，还有印度优秀不错的

不过嘛，发现已养成了记录的习惯
渐渐地发现，竟然看了如此之多（其中，不乏遗漏&跳着看等没有记录的）…
数量是可观的，由其（影视）产生的观念不得不说是由影响的
不知是否和我一样，熬夜也要坚持看完电影的人
。。。。。。
最后，无论看什么电影&影视,推荐还是观影未删减版；如果能力可以，则观看外国大片尽量看原版

在最后：感谢儿时曾伴随我成长的影视，以及逝去&懵懂（少年）童年时光。虽然在当时，可能连个（影视）结局&全部都不能看完的年代，但岁月静好…
